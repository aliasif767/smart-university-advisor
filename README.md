

# Smart University Advisor

**Smart University Advisor** is an advanced solution designed to assist students in planning their academic paths, selecting suitable courses, and obtaining guidance on various academic topics, utilizing a modern web development stack.

---

## 🛠 Technologies Used

- **TypeScript** (75.6%)
- **JavaScript** (21%)
- **Python** (2.9%)
- **Other** (0.5%)

---

## 🚀 Features

- AI-driven course recommendations.
- User-friendly interface for academic planning.
- Cross-language support integrating TypeScript, JavaScript, and Python.

## 🖥️ How to Set Up and Run this Project Locally

Follow the below steps to clone the repository, install dependencies, and run the project.

### Prerequisites

Before running the project, ensure you have the following installed:

- Node.js (v16 or above)
- NPM or Yarn
- Python (for AI components)

---

### Steps

#### 1. Clone the Repository

```bash
git clone https://github.com/aliasif767/smart-university-advisor.git
cd smart-university-advisor
```

---

#### 2. Install Dependencies

Run the following command to install the required packages:

```bash
npm install
```
or, if using Yarn:

```bash
yarn install
```

---

#### 3. Run the Project

Start the project locally using the following script:

```bash
npm run dev
```

For Yarn:

```bash
yarn dev
```

The project should now be running at `http://localhost:3000`.

---

#### 4. Backend Setup (If Applicable)

- If the project has a Python backend, navigate to the Python directory and install backend dependencies:

```bash
# Example commands to install python packages
pip install -r requirements.txt
python3 app.py
```
